# Portfolio David Ruiz

This project is a personal portfolio built with Next.js, Vite, and Tailwind CSS.

![Portfolio Screenshot](demo.png)

## Installation

1. Install dependencies:

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

## Production Preview

```bash
npm run preview
```
